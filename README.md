# Gagan Kumar Reddy — Space Portfolio 🚀

A 3D deep space portfolio built with React + Three.js.

## Features
- Live Three.js 3D scene with 9,000 stars, nebula, and scroll-driven camera fly-through
- Glitch text effect on name
- Cursor light trail
- Warp speed lines on fast scroll
- Ambient space audio (toggleable)
- Animated skill bars, hover project cards, contact form

## Setup & Run

```bash
# 1. Install dependencies
npm install

# 2. Run locally
npm run dev

# 3. Open http://localhost:5173
```

## Deploy to Vercel

```bash
# 1. Push this folder to a GitHub repo
# 2. Go to vercel.com → New Project → Import your repo
# 3. Framework: Vite (auto-detected)
# 4. Hit Deploy ✅
```

## Customize
- Edit personal data in `src/GaganPortfolio.jsx` at the top (PROFILE, PROJECTS, SKILLS, CERTS, ACHIEVEMENTS)
- Add real GitHub links to project cards
- Connect contact form to EmailJS or a backend API
